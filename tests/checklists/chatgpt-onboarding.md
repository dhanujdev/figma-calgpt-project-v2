# ChatGPT Onboarding Checklist

- New conversation teaches example prompts for dashboard, meal logging, weight logging, daily check-in, and weekly review.
- Brand-new user gets onboarding guidance instead of a sparse empty summary.
- Existing user gets direct action and concise follow-up, not a repeated onboarding block.
- Responses do not narrate the same tool action twice.
- The widget reinforces that writes happen in chat and display updates after the agent acts.
